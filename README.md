## Projects

### 1. COVID-19 Vaccine Uptake — Geographic Disparity Analysis
Modelled vaccination inequality across 163 countries using OLS regression 
(R² = 0.497) and K-Means clustering that recovered continental geography 
with 75% accuracy — given zero geographic input.

**Tools:** R, lmtest, ggplot2

---

### 2. Netflix Content Trend Analysis
Analyzed 8,807 titles to surface a content surge peaking in 2019, movies 
comprising 67% of the catalog, and the US, India, and UK as the top 3 
contributing markets.

**Tools:** R, ggplot2, tidyverse

---

### 3. Spam Email Classifier
Benchmarked 3 ML models on 4,601 emails. SVM with RBF kernel achieved 
82% accuracy and 89% specificity after duplicate removal and feature 
selection via permutation importance.

**Tools:** R, caret, e1071
